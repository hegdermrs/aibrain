  Getting Started Guide

STEP 1: Get Your API Key (2 minutes)
────────────────────────────────────

1. Open browser and go to:
   https://console.anthropic.com/

2. Sign up or log in

3. Click "Keys" in left menu

4. Click "Create Key"

5. Copy the key (starts with sk-ant-)

6. Save it somewhere safe (you'll need it in Step 3)


STEP 2: Install Everything (2 minutes)
───────────────────────────────────────

1. Double-click this file:
   INSTALL.bat

2. Wait for the installer window

3. It will say "Setup complete!" when done

That's it! All dependencies installed.


STEP 3: Configure Your API Key (1 minute)
──────────────────────────────────────────

1. Find the file: .env

2. Right-click it → Open with Notepad

3. Find this line:
   ANTHROPIC_API_KEY=

4. Add your key from Step 1:
   ANTHROPIC_API_KEY=sk-ant-xxxxx

5. Save and close


STEP 4: Run the Dashboard (30 seconds)
──────────────────────────────────────

1. Double-click this file:
   START_DASHBOARD.bat

2. Wait for message: "Streamlit ready to go"

3. Your browser opens automatically

4. Dashboard loads at: http://localhost:8501


STEP 5: Test It Out (5 minutes)
───────────────────────────────

The dashboard has sample data built-in!

1. Go to: 📋 Briefing
   - Click "Generate Briefing"
   - See your briefing in tabs

2. Go to: 📞 Analyze Call
   - See sample transcript loaded
   - Click "Analyze"
   - Review decisions & follow-ups

3. Go to: 🔍 Strategic Lens
   - Enter a question: "Should we raise prices?"
   - Click "Run Lens Analysis"
   - See perspectives from different angles

4. Go to: 📚 History
   - See everything you've generated


SUCCESS! You're ready to use the Brain.
